This is the official Fejix website!

Fejix is a cross-platform GUI framework in early stage of development.

Its essence is to create optimized GUI by speifying it in a domain-specific language in a concise and semantically correct way. The language is called Fjml (Fejix Markup Language)

This page contains links to documentation and tutorials. However, they are still incomplete or unwritten.
